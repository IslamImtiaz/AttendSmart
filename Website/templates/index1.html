How It Works

  1. Set up university details.
  2. Enter the Subject Name.
  3. Enter the lecture days.
  4. Enter the date range.
  5. Upload student data in `.txt` format.
  6. Generate and save the Excel attendance sheet.

How to Create a .txt File

Follow these simple steps to create a `.txt` file with your attendance data:

  1. Open any text editor (e.g., Notepad on Windows, TextEdit on Mac).
  2. Write the Registration No., hit the "Tab" key, and then write the Student Name.
    Note: These headings are necessary to add.
  3. Enter the attendance data in the format: "2022-ME-521" Name_of_Student.
    Note: Don’t forget to add a Tab space between the Reg. No. and the Name.
  4. Click on "File" > "Save As".
  5. Choose "All Files" for the file type and name the file with a `.txt` extension (e.g., attendance.txt).
  6. Click "Save" and your file is ready!
